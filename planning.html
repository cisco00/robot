Planning¶

Spline path sampling functions, useful for differentiable continuous robot path planning.

ivy_robot.planning.sample_spline_path(anchor_points, anchor_vals, sample_points, order=3, f=None)[source]¶

Sample spline path, given sample locations, for path defined by the anchor locations and points. [reference]

Parameters

  • anchor_points (array) – Anchor locations between 0-1 (regular spacing not necessary) [batch_shape,num_anchors,1]

  • anchor_vals (array) – Anchor points along the spline path, in path space [batch_shape,num_anchors,path_dim]

  • sample_points (array) – Sample locations between 0-1 [batch_shape,num_samples,1]

  • order (float) – Order of the spline path interpolation

  • f (ml_framework, optional) – Machine learning library. Inferred from inputs if None.

Returns

Spline path sampled at sample_locations, giving points in path space [batch_shape,num_samples,path_dim]
